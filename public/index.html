API Documentation

GET /api/users

Retrieve all users

curl -X GET http://localhost:5000/api/users

GET /api/users/:id

Retrieve a specific user by ID

curl -X GET http://localhost:5000/api/users/user-id

POST /api/users

Create a new user

curl -X POST http://localhost:5000/api/users \
  -H "Content-Type: application/json" \
  -d '{"name":"John","email":"john@example.com","phone":"+1-555-0101"}'

PUT /api/users/:id

Update a user

curl -X PUT http://localhost:5000/api/users/user-id \
  -H "Content-Type: application/json" \
  -d '{"name":"Jane","email":"jane@example.com"}'

DELETE /api/users/:id

Delete a user

curl -X DELETE http://localhost:5000/api/users/user-id

GET /api/users/search?q=query

Search users by name or email

curl -X GET "http://localhost:5000/api/users/search?q=john"
